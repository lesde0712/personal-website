Bootstrap

Is a framework for responsive web development, providing pre-built HTML, CSS, and JavaScript components. It simplifies design and ensures consistent layout across devices.

HTML

Hypertext Markup Language is the standard language for creating web pages. It uses tags to define the structure and content of a webpage, interpreted by web browsers to display text, images, and multimedia.

JavaScript

Is a dynamic scripting language for web development, enabling interactivity and real-time updates. It manipulates HTML and CSS to create dynamic web experiences executed on the client-side.

CSS

Cascading Style Sheets a language that enhances the presentation of HTML and XML documents. It controls visual aspects of HTML elements, improving webpage appearance and user experience.

Python

Is a high-level programming language valued for its simplicity and readability. It enables concise code expression across diverse domains, including web development, data analysis, AI, and automation.
